# Publishing Guide

## Create GitHub Repository

1. Create a new public repository on GitHub.
2. Recommended repository name:

```text
wooseok-game-client-portfolio
```

3. Do not initialize with README because this folder already has one.
4. Push from this local folder:

```bash
git remote add origin https://github.com/<your-id>/wooseok-game-client-portfolio.git
git branch -M main
git push -u origin main
```

## After Publishing

Replace placeholders:

- `TODO_GITHUB_URL` in Notion
- `TODO_NOTION_URL` in GitHub README
- `TODO_PDF_URL` in GitHub README

Then update the PPT/PDF link page with:

- PDF portfolio
- Notion portfolio
- GitHub samples

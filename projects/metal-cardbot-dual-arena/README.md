# 메탈카드봇 듀얼아레나

아케이드 하드웨어 연동 Unity 프로젝트입니다. 메인 개발로 시리얼 통신, RF 카드 인식, 카드 배출/QR 흐름과 게임 진행 연동을 담당했습니다.

## 샘플 코드

- `SerialPortManager_Protocol.cs`: IO 보드 초기화, XML 기반 포트 설정, read/write 버퍼 처리
- `RFCard_DeviceInventory.cs`: ISO15693 RF 카드 태그 인벤토리와 UID 연결 흐름
